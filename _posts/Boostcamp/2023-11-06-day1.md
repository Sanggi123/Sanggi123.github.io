* ## Variable & List
  * 파이썬은 메모리의 주소를 참조한다.

```python
a = [1,2,3,4,5]
a
```

    [1, 2, 3, 4, 5]

```python
b = a # a의 메모리 주소를 갖고옴
a is b
```

    True

```python
b = a[:] # a의 값을 복사함 (copy와 같)
a is b
```

    False
