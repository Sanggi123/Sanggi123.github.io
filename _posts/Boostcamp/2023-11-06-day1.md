* ## Variable & List
  * 파이썬은 메모리의 주소를 참조한다.

```python
a = [1,2,3,4,5]
a
```

    [1, 2, 3, 4, 5]

```python
b = a # a의 메모리 주소를 갖고옴
a is b
```

    True

```python
b = a[:] # a의 값을 복사함 (copy)
a is b
```

    False

- # String and advanced function concept

```python
def list_add_var(list_b):
    list_b.append(1) # 기존 객체의 주소값을 사용
    list_b = [2,3]   # 변수 선언 시 새로운 객체 생성

list_a = [0]
list_add_var(list_a)
print(list_a)
```

# 느낀 점🤔
